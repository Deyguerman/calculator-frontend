# Quasar App (calculator)

A Quasar Project

## Install the dependencies
```bash
yarn
# or
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```

### Add .env file
```
Rename .env.template to .env to enable environment variables.
```


### Lint the files
```bash
yarn lint
# or
npm run lint
```
